# Jagdhunderassen Quiz

Interaktives Lernspiel für die Jägerprüfung in Mecklenburg-Vorpommern.

## Features
- 32 Jagdhunderassen mit Bildern
- Angewölfte Eigenschaften und jagdliche Spezialisierung
- Verschiedene Spielmodi (Alle, Zufällig, Nur Fehler)
- Kategorie-Filter (Vorstehhunde, Schweißhunde, etc.)
- Offline-fähig als PWA

## Installation auf iPhone
1. Öffne die GitHub Pages URL in Safari
2. Tippe auf das Teilen-Symbol
3. Wähle "Zum Home-Bildschirm"

## GitHub Pages aktivieren
1. Gehe zu Settings → Pages
2. Source: Deploy from a branch
3. Branch: main, Ordner: / (root)
4. Save

Die App ist dann unter `https://[username].github.io/jagdhund-quiz/` erreichbar.
